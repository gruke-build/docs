Remarks

This is a CLI wrapper with fluent API that allows to modify the following arguments:

  • <stack> via Stack
  • --config via Config
  • --debug via Debug
  • --filter via Filter
  • --format via Format
  • --kubeconfig via Kubeconfig
  • --log-level via LogLevel
  • --namespace via Namespace
  • --no-resolve via NoResolve
  • --no-trunc via NoTrunc
  • --orchestrator via Orchestrator
  • --quiet via Quiet
  • --tls via TLS
  • --tlscacert via TLSCaCert
  • --tlscert via TLSCert
  • --tlskey via TLSKey
  • --tlsverify via TLSVerify
