Class DotNetWorkloadUpdateSettings

Namespace
Nuke.Common.Tools.DotNet
Assembly
Nuke.Common.dll

The dotnet workload update command updates all installed workloads to the newest available versions. It queries Nuget.org for updated workload manifests. It then updates local manifests, downloads new versions of the installed workloads, and removes all old versions of each workload.

For more details, visit the official website.

Remarks

This is a CLI wrapper with fluent API that allows to modify the following arguments:

  • --advertising-manifests-only via AdvertisingManifestsOnly
  • --configFile via ConfigFile
  • --disable-parallel via DisableParallel
  • --from-previous-sdk via FromPreviousSdk
  • --ignore-failed-sources via IgnoreFailedSources
  • --include-previews via IncludePreviews
  • --interactive via Interactive
  • --no-cache via NoCache
  • --skip-sign-check via SkipSignCheck
  • --source via Source
  • --temp-dir via TempDir
  • --verbosity via Verbosity
