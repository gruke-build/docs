Remarks

This is a CLI wrapper with fluent API that allows to modify the following arguments:

  • --file-type via FileType
  • --preview via Preview
  • --read-write-timeout via ReadWriteTimeout
  • --retries via Retries
  • --stable via Stable
  • --timeout via Timeout
  • --url via Url
  • --version via Version
