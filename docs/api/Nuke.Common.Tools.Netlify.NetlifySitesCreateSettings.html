Remarks

This is a CLI wrapper with fluent API that allows to modify the following arguments:

  • --account-slug via AccountSlug
  • --debug via Debug
  • --httpProxy via HttpProxy
  • --httpProxyCertificateFilename via HttpProxyCertificateFileName
  • --manual via Manual
  • --name via Name
  • --with-ci via WithCI
