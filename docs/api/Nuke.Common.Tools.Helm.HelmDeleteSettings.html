Class HelmDeleteSettings

Namespace
Nuke.Common.Tools.Helm
Assembly
Nuke.Common.dll

This command takes a release name, and then deletes the release from Kubernetes. It removes all of the resources associated with the last release of the chart. Use the '--dry-run' flag to see which releases will be deleted without actually deleting them.

For more details, visit the official website.

Remarks

This is a CLI wrapper with fluent API that allows to modify the following arguments:

  • <releaseNames> via ReleaseNames
  • --debug via Debug
  • --description via Description
  • --dry-run via DryRun
  • --home via Home
  • --host via Host
  • --kube-context via KubeContext
  • --kubeconfig via Kubeconfig
  • --no-hooks via NoHooks
  • --purge via Purge
  • --tiller-connection-timeout via TillerConnectionTimeout
  • --tiller-namespace via TillerNamespace
  • --timeout via Timeout
  • --tls via Tls
  • --tls-ca-cert via TlsCaCert
  • --tls-cert via TlsCert
  • --tls-hostname via TlsHostname
  • --tls-key via TlsKey
  • --tls-verify via TlsVerify
