Remarks

This is a CLI wrapper with fluent API that allows to modify the following arguments:

  • /Assembly via Assembly
  • /AssemblyConfig via AssemblyConfig
  • /File via File
  • /ReferencePaths via ReferencePaths
  • /UseNuGetCache via UseNuGetCache
  • /Variables via Variables
