Pulumi works with traditional infrastructure like VMs, networks, and databases, in addition to modern architectures, including containers, Kubernetes clusters, and serverless functions. Pulumi supports dozens of public, private, and hybrid cloud service providers.

Remarks

This is a CLI wrapper with fluent API that allows to modify the following arguments:

  • <stackName> via StackName
  • --color via Color
  • --create via Create
  • --cwd via Cwd
  • --disable-integrity-checking via DisableIntegrityChecking
  • --emoji via Emoji
  • --logflow via LogFlow
  • --logtostderr via LogToStderr
  • --non-interactive via NonInteractive
  • --profiling via Profiling
  • --secrets-provider via SecretsProvider
  • --tracing via Tracing
  • --verbose via Verbose
