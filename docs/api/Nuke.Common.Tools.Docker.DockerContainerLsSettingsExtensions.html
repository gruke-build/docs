Remarks

This is a CLI wrapper with fluent API that allows to modify the following arguments:

  • --all via All
  • --config via Config
  • --debug via Debug
  • --filter via Filter
  • --format via Format
  • --last via Last
  • --latest via Latest
  • --log-level via LogLevel
  • --no-trunc via NoTrunc
  • --quiet via Quiet
  • --size via Size
  • --tls via TLS
  • --tlscacert via TLSCaCert
  • --tlscert via TLSCert
  • --tlskey via TLSKey
  • --tlsverify via TLSVerify
