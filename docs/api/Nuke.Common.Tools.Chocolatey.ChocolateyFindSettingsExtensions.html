Remarks

This is a CLI wrapper with fluent API that allows to modify the following arguments:

  • <filter> via Filter
  • --accept-license via AcceptLicense
  • --all-versions via AllVersions
  • --allow-unofficial-build via AllowUnofficialBuild
  • --approved-only via ApprovedOnly
  • --by-id-only via ByIdOnly
  • --by-tag-only via ByTagOnly
  • --cache-location via CacheLocation
  • --cert via ClientCertificate
  • --certpassword via CertificatePassword
  • --confirm via Confirm
  • --debug via Debug
  • --detailed via Detailed
  • --disable-package-repository-optimizations via DisablePackageRepositoryOptimizations
  • --download-cache-only via DownloadCacheAvailable
  • --exact via Exact
  • --execution-timeout via CommandExecutionTimeout
  • --fail-on-standard-error via FailOnStandardError
  • --force via Force
  • --id-only via IdOnly
  • --id-starts-with via IdStartsWith
  • --include-programs via IncludePrograms
  • --limit-output via LimitOutput
  • --local-only via LocalOnly
  • --log-file via LogFile
  • --no-color via NoColor
  • --no-progress via DoNotShowProgress
  • --not-broken via NotBroken
  • --order-by-popularity via OrderByPopularity
  • --page via Page
  • --page-size via PageSize
  • --password via Password
  • --prerelease via Prerelease
  • --proxy via Proxy
  • --proxy-bypass-list via ProxyBypassList
  • --proxy-bypass-on-local via ProxyBypassOnLocal
  • --proxy-password via ProxyPassword
  • --proxy-user via ProxyUserName
  • --show-audit-info via ShowAuditInformation
  • --source via Source
  • --trace via Trace
  • --use-system-powershell via UseSystemPowerShell
  • --user via User
  • --verbose via Verbose
  • --version via Version
