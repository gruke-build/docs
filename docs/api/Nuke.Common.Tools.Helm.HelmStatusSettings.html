Class HelmStatusSettings

Namespace
Nuke.Common.Tools.Helm
Assembly
Nuke.Common.dll

This command shows the status of a named release. The status consists of: - last deployment time - k8s namespace in which the release lives - state of the release (can be: UNKNOWN, DEPLOYED, DELETED, SUPERSEDED, FAILED or DELETING) - list of resources that this release consists of, sorted by kind - details on last test suite run, if applicable - additional notes provided by the chart.

For more details, visit the official website.

Remarks

This is a CLI wrapper with fluent API that allows to modify the following arguments:

  • <releaseName> via ReleaseName
  • --debug via Debug
  • --home via Home
  • --host via Host
  • --kube-context via KubeContext
  • --kubeconfig via Kubeconfig
  • --output via Output
  • --revision via Revision
  • --tiller-connection-timeout via TillerConnectionTimeout
  • --tiller-namespace via TillerNamespace
  • --tls via Tls
  • --tls-ca-cert via TlsCaCert
  • --tls-cert via TlsCert
  • --tls-hostname via TlsHostname
  • --tls-key via TlsKey
  • --tls-verify via TlsVerify
