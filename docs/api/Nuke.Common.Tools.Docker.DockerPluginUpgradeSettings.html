Remarks

This is a CLI wrapper with fluent API that allows to modify the following arguments:

  • <plugin> via Plugin
  • <remote> via Remote
  • --config via Config
  • --debug via Debug
  • --disable-content-trust via DisableContentTrust
  • --grant-all-permissions via GrantAllPermissions
  • --log-level via LogLevel
  • --skip-remote-check via SkipRemoteCheck
  • --tls via TLS
  • --tlscacert via TLSCaCert
  • --tlscert via TLSCert
  • --tlskey via TLSKey
  • --tlsverify via TLSVerify
