Remarks

This is a CLI wrapper with fluent API that allows to modify the following arguments:

  • -ConfigFile via ConfigFile
  • -ForceEnglishOutput via ForceEnglishOutput
  • -Name via Name
  • -NonInteractive via NonInteractive
  • -Password via Password
  • -Source via Source
  • -StorePasswordInClearText via StorePasswordInClearText
  • -UserName via UserName
  • -Verbosity via Verbosity
