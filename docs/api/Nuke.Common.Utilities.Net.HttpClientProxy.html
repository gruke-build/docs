Class HttpClientProxy

Namespace
Nuke.Common.Utilities.Net

Nuke.Utilities.Net.dll

A simple wrapper around HttpClient which calls the specified callback with (non-private) information about all requests issued via this class.
This class is non-owning of the passed HttpClient. The caller is responsible for disposal, if needed.
