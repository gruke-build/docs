Remarks

This is a CLI wrapper with fluent API that allows to modify the following arguments:

  • --additional-categories via AdditionalCategories
  • --appId via AppId
  • --application-name via ApplicationName
  • --directory via Directory
  • --homepage via Homepage
  • --icon via Icon
  • --license via License
  • --main-category via MainCategory
  • --output via Output
  • --screenshot-urls via ScreenshotUrls
  • --summary via Summary
  • --version via Version
