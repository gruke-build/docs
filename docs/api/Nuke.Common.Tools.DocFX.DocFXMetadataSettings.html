Remarks

This is a CLI wrapper with fluent API that allows to modify the following arguments:

  • <projects> via Projects
  • --correlationId via CorrelationId
  • --disableDefaultFilter via DisableDefaultFilter
  • --disableGitFeatures via DisableGitFeatures
  • --filter via FilterConfigFile
  • --force via ForceRebuild
  • --globalNamespaceId via GlobalNamespaceId
  • --help via PrintHelpMessage
  • --log via LogFilePath
  • --logLevel via LogLevel
  • --output via OutputFolder
  • --property via MSBuildProperties
  • --raw via PreserveRawInlineComments
  • --repositoryRoot via RepoRoot
  • --shouldSkipMarkup via ShouldSkipMarkup
  • --warningsAsErrors via WarningsAsErrors
