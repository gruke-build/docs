Remarks

This is a CLI wrapper with fluent API that allows to modify the following arguments:

  • --apiGlobPattern via ApiSourceGlobPattern
  • --apiSourceFolder via ApiSourceFolder
  • --file via OnlyConfigFile
  • --help via PrintHelpMessage
  • --output via OutputFolder
  • --overwrite via Overwrite
  • --quiet via Quiet
