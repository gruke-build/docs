Remarks

This is a CLI wrapper with fluent API that allows to modify the following arguments:

  • <assemblies> via Assemblies
  • --appveyor via AppVeyor
  • --html via HtmlOutput
  • --no-appveyor-autodetect via NoAppVeyor
  • --no-color via NoColor
  • --no-teamcity-autodetect via NoTeamCity
  • --progress via DottedProgress
  • --silent via Silent
  • --teamcity via TeamCity
  • --timeinfo via TimeInfo
  • --xml via XmlOutput
  • -exclude via Excludes
  • -filters via Filters
  • -include via Includes
