Remarks

This is a CLI wrapper with fluent API that allows to modify the following arguments:

  • <container> via Container
  • <network> via Network
  • --alias via Alias
  • --config via Config
  • --debug via Debug
  • --driver-opt via DriverOpt
  • --ip via Ip
  • --ip6 via Ip6
  • --link via Link
  • --link-local-ip via LinkLocalIp
  • --log-level via LogLevel
  • --tls via TLS
  • --tlscacert via TLSCaCert
  • --tlscert via TLSCert
  • --tlskey via TLSKey
  • --tlsverify via TLSVerify
