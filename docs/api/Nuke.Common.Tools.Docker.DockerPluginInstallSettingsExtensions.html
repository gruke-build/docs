Remarks

This is a CLI wrapper with fluent API that allows to modify the following arguments:

  • <keyValues> via KeyValues
  • <plugin> via Plugin
  • --alias via Alias
  • --config via Config
  • --debug via Debug
  • --disable via Disable
  • --disable-content-trust via DisableContentTrust
  • --grant-all-permissions via GrantAllPermissions
  • --log-level via LogLevel
  • --tls via TLS
  • --tlscacert via TLSCaCert
  • --tlscert via TLSCert
  • --tlskey via TLSKey
  • --tlsverify via TLSVerify
