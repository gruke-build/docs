Class HelmDependencyBuildSettingsExtensions

Namespace
Nuke.Common.Tools.Helm
Assembly
Nuke.Common.dll

Build out the charts/ directory from the requirements.lock file. Build is used to reconstruct a chart's dependencies to the state specified in the lock file. This will not re-negotiate dependencies, as 'helm dependency update' does. If no lock file is found, 'helm dependency build' will mirror the behavior of 'helm dependency update'.

For more details, visit the official website.

Remarks

This is a CLI wrapper with fluent API that allows to modify the following arguments:

  • <chart> via Chart
  • --debug via Debug
  • --home via Home
  • --host via Host
  • --keyring via Keyring
  • --kube-context via KubeContext
  • --kubeconfig via Kubeconfig
  • --tiller-connection-timeout via TillerConnectionTimeout
  • --tiller-namespace via TillerNamespace
  • --verify via Verify
