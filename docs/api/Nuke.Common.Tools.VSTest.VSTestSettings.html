Remarks

This is a CLI wrapper with fluent API that allows to modify the following arguments:

  • <testAssemblies> via TestAssemblies
  • /Diag via DiagnosticsFile
  • /EnableCodeCoverage via EnableCodeCoverage
  • /Framework via Framework
  • /InIsolation via InIsolation
  • /ListDiscoverers via ListDiscoverers
  • /ListExecutors via ListExecutors
  • /ListLoggers via ListLoggers
  • /ListSettingsProviders via ListSettingsProviders
  • /ListTests via ListTests
  • /Logger via Logger
  • /Parallel via Parallel
  • /Platform via Platform
  • /Settings via SettingsFile
  • /TestAdapterPath via TestAdapterPath
  • /TestCaseFilter via TestCaseFilters
  • /Tests via Tests
  • /UseVsixExtensions via UseVsixExtensions
