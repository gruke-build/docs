Class EntityFrameworkMigrationsRemoveSettingsExtensions

Namespace
Nuke.Common.Tools.EntityFramework
Assembly
Nuke.Common.dll

The dotnet-ef migrations remove command is used to remove the last migration (rolls back the code changes that were done for the migration).

For more details, visit the official website.

Remarks

This is a CLI wrapper with fluent API that allows to modify the following arguments:

  • --configuration via Configuration
  • --context via Context
  • --force via Force
  • --framework via Framework
  • --json via Json
  • --no-build via NoBuild
  • --no-color via NoColor
  • --prefix-output via PrefixOutput
  • --project via Project
  • --runtime via Runtime
  • --startup-project via StartupProject
  • --verbose via Verbose
