Class NpmInstallSettings

Namespace
Nuke.Common.Tools.Npm
Assembly
Nuke.Common.dll

Installs a package, and any packages that it depends on. If the package has a package-lock or shrinkwrap file, the installation of dependencies will be driven by that, with an npm-shrinkwrap.json taking precedence if both files exist. See package-lock.json and npm-shrinkwrap.

A package is:
  • a) A folder containing a program described by a package.json file
  • b) A gzipped tarball containing (b)
  • c) A url that resolves to (b)
  • d) a <name>@<version> that is published on the registry (see npm-registry) with (c)
  • e) a <name>@<tag> (see npm-dist-tag) that points to (d)
  • f) a <name> that has a "latest" tag satisfying (e)
  • g) a <git remote url> that resolves to (a)

For more details, visit the official website.

Remarks

This is a CLI wrapper with fluent API that allows to modify the following arguments:

  • <packages> via Packages
  • --force via Force
  • --global via Global
  • --global-style via GlobalStyle
  • --ignore-scripts via IgnoreScripts
  • --legacy-bundling via LegacyBundling
  • --link via Link
  • --no-bin-links via NoBinLinks
  • --no-optional via NoOptional
  • --no-shrinkwrap via NoShrinkWrap
  • --nodedir via NodeDir
  • --only via Only
  • --production via Production

GlobalStyle

Causes npm to install the package into your local node_modules folder with the same layout it uses with the global node_modules folder. Only your direct dependencies will show in node_modules and everything they depend on will be flattened in their node_modules folders. This obviously will eliminate some deduping.

public bool? GlobalStyle { get; }

Property Value

bool?

IgnoreScripts

Causes npm to not execute any scripts defined in the package.json. See npm-scripts.

public bool? IgnoreScripts { get; }

bool?

LegacyBundling

Causes npm to install the package such that versions of npm prior to 1.4, such as the one included with node 0.8, can install the package. This eliminates all automatic deduping.

public bool? LegacyBundling { get; }
