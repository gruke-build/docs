Remarks

This is a CLI wrapper with fluent API that allows to modify the following arguments:

  • <targetPath> via TargetPath
  • --api-key via ApiKey
  • --disable-buffering via DisableBuffering
  • --force-english-output via ForceEnglishOutput
  • --no-service-endpoint via NoServiceEndpoint
  • --no-symbols via NoSymbols
  • --skip-duplicate via SkipDuplicate
  • --source via Source
  • --symbol-api-key via SymbolApiKey
  • --symbol-source via SymbolSource
  • --timeout via Timeout
