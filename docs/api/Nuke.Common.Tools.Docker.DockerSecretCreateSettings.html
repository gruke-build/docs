Remarks

This is a CLI wrapper with fluent API that allows to modify the following arguments:

  • <file> via File
  • <secret> via Secret
  • --config via Config
  • --debug via Debug
  • --driver via Driver
  • --label via Label
  • --log-level via LogLevel
  • --template-driver via TemplateDriver
  • --tls via TLS
  • --tlscacert via TLSCaCert
  • --tlscert via TLSCert
  • --tlskey via TLSKey
  • --tlsverify via TLSVerify
