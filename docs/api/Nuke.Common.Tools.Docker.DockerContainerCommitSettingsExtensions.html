Remarks

This is a CLI wrapper with fluent API that allows to modify the following arguments:

  • <container> via Container
  • <repository> via Repository
  • --author via Author
  • --change via Change
  • --config via Config
  • --debug via Debug
  • --log-level via LogLevel
  • --message via Message
  • --pause via Pause
  • --tls via TLS
  • --tlscacert via TLSCaCert
  • --tlscert via TLSCert
  • --tlskey via TLSKey
  • --tlsverify via TLSVerify
