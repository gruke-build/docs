Remarks

This is a CLI wrapper with fluent API that allows to modify the following arguments:

  • <container> via Container
  • --config via Config
  • --debug via Debug
  • --details via Details
  • --follow via Follow
  • --log-level via LogLevel
  • --since via Since
  • --tail via Tail
  • --timestamps via Timestamps
  • --tls via TLS
  • --tlscacert via TLSCaCert
  • --tlscert via TLSCert
  • --tlskey via TLSKey
  • --tlsverify via TLSVerify
  • --until via Until
