Remarks

This is a CLI wrapper with fluent API that allows to modify the following arguments:

  • <args> via Args
  • <command> via Command
  • <container> via Container
  • --config via Config
  • --debug via Debug
  • --detach via Detach
  • --detach-keys via DetachKeys
  • --env via Env
  • --interactive via Interactive
  • --log-level via LogLevel
  • --privileged via Privileged
  • --tls via TLS
  • --tlscacert via TLSCaCert
  • --tlscert via TLSCert
  • --tlskey via TLSKey
  • --tlsverify via TLSVerify
  • --tty via Tty
  • --user via User
  • --workdir via Workdir
