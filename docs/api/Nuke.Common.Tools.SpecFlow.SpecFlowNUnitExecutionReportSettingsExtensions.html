Class SpecFlowNUnitExecutionReportSettingsExtensions

Namespace
Nuke.Common.Tools.SpecFlow
Assembly
Nuke.Common.dll

This report provides a formatted HTML report of a test execution. The report contains a summary about the executed tests and the result and also a detailed report for the individual scenario executions.

For more details, visit the official website.

Remarks

This is a CLI wrapper with fluent API that allows to modify the following arguments:

  • --FeatureLanguage via FeatureLanguage
  • --OutputFile via OutputFile
  • --ProjectFile via ProjectFile
  • --ProjectName via ProjectName
  • --testOutput via TestOutput
  • --xmlTestResult via XmlTestResult
  • --XsltFile via XsltFile
