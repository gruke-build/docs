Remarks

This is a CLI wrapper with fluent API that allows to modify the following arguments:

  • /AdditionalContractNamespaceUsages via AdditionalContractNamespaceUsages
  • /AdditionalNamespaceUsages via AdditionalNamespaceUsages
  • /AnyType via AnyType
  • /ArrayBaseType via ArrayBaseType
  • /ArrayInstanceType via ArrayInstanceType
  • /ArrayType via ArrayType
  • /ClassName via ClassName
  • /ClassStyle via ClassStyle
  • /ClientBaseClass via ClientBaseClass
  • /ClientClassAccessModifier via ClientClassAccessModifier
  • /ConfigurationClass via ConfigurationClass
