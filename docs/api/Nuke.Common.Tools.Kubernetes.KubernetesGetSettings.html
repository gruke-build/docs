Class KubernetesGetSettings

Namespace
Nuke.Common.Tools.Kubernetes
Assembly
Nuke.Common.dll

Display one or many resources Prints a table of the most important information about the specified resources. You can filter the list using a label selector and the --selector flag. If the desired resource type is namespaced you will only see results in your current namespace unless you pass --all-namespaces. Uninitialized objects are not shown unless --include-uninitialized is passed. By specifying the output as 'template' and providing a Go template as the value of the --template flag, you can filter the attributes of the fetched resources.Use "kubectl api-resources" for a complete list of supported resources.

For more details, visit the official website.

Remarks

This is a CLI wrapper with fluent API that allows to modify the following arguments:

  • <typeName> via TypeName
  • --all-namespaces via AllNamespaces
  • --allow-missing-template-keys via AllowMissingTemplateKeys
  • --alsologtostderr via Alsologtostderr
  • --as via As
  • --as-group via AsGroup
  • --cache-dir via CacheDir
  • --certificate-authority via CertificateAuthority
  • --chunk-size via ChunkSize
  • --client-certificate via ClientCertificate
  • --client-key via ClientKey
  • --cluster via Cluster
  • --context via Context
  • --export via Export
  • --field-selector via FieldSelector
  • --filename via Filename
  • --ignore-not-found via IgnoreNotFound
  • --include-uninitialized via IncludeUninitialized
  • --insecure-skip-tls-verify via InsecureSkipTlsVerify
  • --kubeconfig via Kubeconfig
  • --label-columns via LabelColumns
  • --log-backtrace-at via LogBacktraceAt
  • --log-dir via LogDir
  • --logtostderr via Logtostderr
  • --match-server-version via MatchServerVersion
  • --namespace via Namespace
  • --no-headers via NoHeaders
  • --output via Output
  • --raw via Raw
  • --recursive via Recursive
  • --request-timeout via RequestTimeout
  • --selector via Selector
  • --server via Server
  • --server-print via ServerPrint
  • --show-all via ShowAll
  • --show-kind via ShowKind
  • --show-labels via ShowLabels
  • --sort-by via SortBy
  • --stderrthreshold via Stderrthreshold
  • --template via Template
  • --token via Token
  • --use-openapi-print-columns via UseOpenapiPrintColumns
  • --user via User
  • --v via V
  • --vmodule via Vmodule
  • --watch via Watch
  • --watch-only via WatchOnly
