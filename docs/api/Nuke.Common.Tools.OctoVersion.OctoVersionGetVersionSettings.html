Remarks

This is a CLI wrapper with fluent API that allows to modify the following arguments:

  • --BuildMetadata via BuildMetadata
  • --CurrentBranch via CurrentBranch
  • --DetectEnvironment via DetectEnvironment
  • --FullSemVer via FullSemVer
  • --Major via Major
  • --Minor via Minor
  • --NonPreReleaseTags via NonPreReleaseTags
  • --OutputFormats via OutputFormats
  • --OutputJsonFile via OutputJsonFile
  • --Patch via Patch
  • --PreReleaseTag via PreReleaseTag
  • --RepositoryPath via RepositoryPath
