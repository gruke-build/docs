Remarks

This is a CLI wrapper with fluent API that allows to modify the following arguments:

  • <scriptFile> via ScriptFile
  • /D via KeyValueDefinitions
  • /D via KeyDefinitions
  • /F via OutputBaseFilename
  • /O via Output
  • /O via OutputDir
  • /Q via Quiet
  • /Qp via QuietWithProgress
  • /S via SignTools
  • /V via Verbosity
