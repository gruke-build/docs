Class KubernetesDescribeSettingsExtensions

Namespace
Nuke.Common.Tools.Kubernetes
Assembly
Nuke.Common.dll

Show details of a specific resource or group of resources Print a detailed description of the selected resources, including related resources such as events or controllers. You may select a single object by name, all objects of that type, provide a name prefix, or label selector. For example: $ kubectl describe TYPE NAME_PREFIX will first check for an exact match on TYPE and NAME PREFIX. If no such resource exists, it will output details for every resource that has a name prefixed with NAME PREFIX.Use "kubectl api-resources" for a complete list of supported resources.

For more details, visit the official website.

Remarks

This is a CLI wrapper with fluent API that allows to modify the following arguments:

  • <typeName> via TypeName
  • --all-namespaces via AllNamespaces
  • --alsologtostderr via Alsologtostderr
  • --as via As
  • --as-group via AsGroup
  • --cache-dir via CacheDir
  • --certificate-authority via CertificateAuthority
  • --client-certificate via ClientCertificate
  • --client-key via ClientKey
  • --cluster via Cluster
  • --context via Context
  • --filename via Filename
  • --include-uninitialized via IncludeUninitialized
  • --insecure-skip-tls-verify via InsecureSkipTlsVerify
  • --kubeconfig via Kubeconfig
  • --log-backtrace-at via LogBacktraceAt
  • --log-dir via LogDir
  • --logtostderr via Logtostderr
  • --match-server-version via MatchServerVersion
  • --namespace via Namespace
  • --recursive via Recursive
  • --request-timeout via RequestTimeout
  • --selector via Selector
  • --server via Server
  • --show-events via ShowEvents
  • --stderrthreshold via Stderrthreshold
  • --token via Token
  • --user via User
  • --v via V
  • --vmodule via Vmodule
