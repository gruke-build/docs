Remarks

This is a CLI wrapper with fluent API that allows to modify the following arguments:

  • --append via Append
  • --bootstrap via Bootstrap
  • --buildkitd-flags via BuildKitDaemonFlags
  • --buildkitd-flags via BuildKitDFlags
  • --config via Config
  • --config via Config
  • --debug via Debug
  • --driver via Driver
  • --driver-opt via DriverOpt
  • --leave via Leave
  • --log-level via LogLevel
  • --name via Name
  • --node via Node
  • --platform via Platform
  • --tls via TLS
  • --tlscacert via TLSCaCert
  • --tlscert via TLSCert
  • --tlskey via TLSKey
  • --tlsverify via TLSVerify
  • --use via Use
