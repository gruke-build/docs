Pulumi is an open source infrastructure as code tool for creating, deploying and managing cloud infrastructure. Pulumi works with traditional infrastructure like VMs, networks, and databases, in addition to modern architectures, including containers, Kubernetes clusters, and serverless functions. Pulumi supports dozens of public, private, and hybrid cloud service providers.

For more details, visit the official website.

Remarks

This is a CLI wrapper with fluent API that allows to modify the following arguments:

  • <template> via Template
  • --color via Color
  • --config via Config
  • --config-path via ConfigPath
  • --cwd via Cwd
  • --description via Description
  • --dir via Directory
  • --disable-integrity-checking via DisableIntegrityChecking
  • --emoji via Emoji
  • --force via Force
  • --generate-only via GenerateOnly
  • --logflow via LogFlow
  • --logtostderr via LogToStderr
  • --name via Name
  • --non-interactive via NonInteractive
  • --offline via Offline
  • --profiling via Profiling
  • --secrets-provider via SecretsProvider
  • --stack via Stack
  • --tracing via Tracing
  • --verbose via Verbose
  • --yes via Yes
