Remarks

This is a CLI wrapper with fluent API that allows to modify the following arguments:

  • <chart> via Chart
  • --ca-file via CaFile
  • --cert-file via CertFile
  • --debug via Debug
  • --devel via Devel
  • --home via Home
  • --host via Host
  • --key-file via KeyFile
  • --keyring via Keyring
  • --kube-context via KubeContext
  • --kubeconfig via Kubeconfig
  • --password via Password
  • --repo via Repo
  • --tiller-connection-timeout via TillerConnectionTimeout
  • --tiller-namespace via TillerNamespace
  • --username via Username
  • --verify via Verify
  • --version via Version
