Pulumi supports dozens of public, private, and hybrid cloud service providers.

Remarks

This is a CLI wrapper with fluent API that allows to modify the following arguments:

  • --all via All
  • --color via Color
  • --cwd via Cwd
  • --disable-integrity-checking via DisableIntegrityChecking
  • --emoji via Emoji
  • --json via Json
  • --logflow via LogFlow
  • --logtostderr via LogToStderr
  • --non-interactive via NonInteractive
  • --organization via Organization
  • --profiling via Profiling
  • --project via Project
  • --tag via Tag
  • --tracing via Tracing
  • --verbose via Verbose
