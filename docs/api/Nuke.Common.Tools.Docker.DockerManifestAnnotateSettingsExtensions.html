Remarks

This is a CLI wrapper with fluent API that allows to modify the following arguments:

  • <manifest> via Manifest
  • <manifestList> via ManifestList
  • --arch via Arch
  • --config via Config
  • --debug via Debug
  • --log-level via LogLevel
  • --os via Os
  • --os-features via OsFeatures
  • --tls via TLS
  • --tlscacert via TLSCaCert
  • --tlscert via TLSCert
  • --tlskey via TLSKey
  • --tlsverify via TLSVerify
  • --variant via Variant
