Class OctopusPackSettingsExtensions

Namespace
Nuke.Common.Tools.Octopus
Assembly
Nuke.Common.dll

The Octo.exe pack command provides a number of other useful parameters that can be used to customize the way your package gets created, such as output folder, files to include and release notes.

For more details, visit the official website.

Remarks

This is a CLI wrapper with fluent API that allows to modify the following arguments:

  • --author via Authors
  • --basePath via BasePath
  • --description via Description
  • --format via Format
  • --id via Id
  • --include via Include
  • --outFolder via OutputFolder
  • --overwrite via Overwrite
  • --releaseNotes via ReleaseNotes
  • --releaseNotesFile via ReleaseNotesFile
  • --title via Title
  • --verbose via Verbose
  • --version via Version
