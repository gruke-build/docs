xUnit.net is a free, open source, community-focused unit testing tool for the .NET Framework. Written by the original inventor of NUnit v2, xUnit.net is the latest technology for unit testing C#, F#, VB.NET and other .NET languages. xUnit.net works with ReSharper, CodeRush, TestDriven.NET and Xamarin. It is part of the .NET Foundation, and operates under their code of conduct. It is licensed under Apache 2 (an OSI approved license).

For more details, visit the official website.

Remarks

This is a CLI wrapper with fluent API that allows to modify the following arguments:

  • <targetAssemblyWithConfigs> via TargetAssemblyWithConfigs
  • - via Reporter
  • - via ResultReports
  • -appdomains via AppDomainMode
  • -class via Classes
  • -debug via Debug
  • -diagnostics via Diagnostics
  • -failskips via FailSkips
  • -maxthreads via MaxThreads
  • -method via Methods
  • -namespace via Namespaces
  • -noautoreporters via NoAutoReporters
  • -nocolor via NoColor
  • -nologo via NoLogo
  • -noshadow via NoShadowCopying
  • -notrait via ExcludedTraits
  • -parallel via Parallel
  • -pause via Pause
  • -serialize via Serialization
  • -stoponfail via StopOnFail
  • -trait via Traits
  • -wait via Wait
