Remarks

This is a CLI wrapper with fluent API that allows to modify the following arguments:

  • <config> via Config
  • <file> via File
  • --config via Config
  • --debug via Debug
  • --label via Label
  • --log-level via LogLevel
  • --template-driver via TemplateDriver
  • --tls via TLS
  • --tlscacert via TLSCaCert
  • --tlscert via TLSCert
  • --tlskey via TLSKey
  • --tlsverify via TLSVerify
