Remarks

This is a CLI wrapper with fluent API that allows to modify the following arguments:

  • <network> via Network
  • --attachable via Attachable
  • --aux-address via AuxAddress
  • --config via Config
  • --config-from via ConfigFrom
  • --config-only via ConfigOnly
  • --debug via Debug
  • --driver via Driver
  • --gateway via Gateway
  • --ingress via Ingress
  • --internal via Internal
  • --ip-range via IpRange
  • --ipam-driver via IpamDriver
  • --ipam-opt via IpamOpt
  • --ipv6 via Ipv6
  • --label via Label
  • --log-level via LogLevel
  • --opt via Opt
  • --scope via Scope
  • --subnet via Subnet
  • --tls via TLS
  • --tlscacert via TLSCaCert
  • --tlscert via TLSCert
  • --tlskey via TLSKey
  • --tlsverify via TLSVerify
