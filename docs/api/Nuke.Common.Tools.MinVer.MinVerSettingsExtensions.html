Remarks

This is a CLI wrapper with fluent API that allows to modify the following arguments:

  • --auto-increment via AutoIncrement
  • --build-metadata via BuildMetadata
  • --default-pre-release-identifiers via DefaultPreReleaseIdentifiers
  • --minimum-major-minor via MinimumMajorMinor
  • --tag-prefix via TagPrefix
  • --verbosity via Verbosity
