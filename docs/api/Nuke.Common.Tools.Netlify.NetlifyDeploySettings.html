Remarks

This is a CLI wrapper with fluent API that allows to modify the following arguments:

  • --alias via Alias
  • --auth via Auth
  • --branch via Branch
  • --build via Build
  • --debug via Debug
  • --dir via Directory
  • --functions via Functions
  • --httpProxy via HttpProxy
  • --httpProxyCertificateFilename via HttpProxyCertificateFileName
  • --json via Json
  • --message via Message
  • --open via Open
  • --prod via Production
  • --prodIfUnlocked via ProductionIfUnlocked
  • --site via Site
  • --timeout via Timeout
  • --trigger via Trigger

Properties

Alias

Specifies the alias for deployment, the string at the beginning of the deploy subdomain. Useful for creating predictable deployment URLs. Avoid setting an alias string to the same value as a deployed branch.

alias doesn’t

create a branch deploy and can’t be used in conjunction with the branch subdomain feature. Maximum 37 characters.
