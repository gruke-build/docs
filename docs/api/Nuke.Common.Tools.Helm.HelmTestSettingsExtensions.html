Class HelmTestSettingsExtensions

Namespace
Nuke.Common.Tools.Helm
Assembly
Nuke.Common.dll

The test command runs the tests for a release. The argument this command takes is the name of a deployed release. The tests to be run are defined in the chart that was installed.

For more details, visit the official website.

Remarks

This is a CLI wrapper with fluent API that allows to modify the following arguments:

  • <release> via Release
  • --cleanup via Cleanup
  • --debug via Debug
  • --home via Home
  • --host via Host
  • --kube-context via KubeContext
  • --kubeconfig via Kubeconfig
  • --parallel via Parallel
  • --tiller-connection-timeout via TillerConnectionTimeout
  • --tiller-namespace via TillerNamespace
  • --timeout via Timeout
  • --tls via Tls
  • --tls-ca-cert via TlsCaCert
  • --tls-cert via TlsCert
  • --tls-hostname via TlsHostname
  • --tls-key via TlsKey
  • --tls-verify via TlsVerify
