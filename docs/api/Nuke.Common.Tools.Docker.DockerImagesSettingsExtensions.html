Remarks

This is a CLI wrapper with fluent API that allows to modify the following arguments:

  • <repository> via Repository
  • --all via All
  • --config via Config
  • --debug via Debug
  • --digests via Digests
  • --filter via Filter
  • --format via Format
  • --log-level via LogLevel
  • --no-trunc via NoTrunc
  • --quiet via Quiet
  • --tls via TLS
  • --tlscacert via TLSCaCert
  • --tlscert via TLSCert
  • --tlskey via TLSKey
  • --tlsverify via TLSVerify
