Class DotNetNuGetDeleteSettingsExtensions

Namespace
Nuke.Common.Tools.DotNet
Assembly
Nuke.Common.dll

The dotnet nuget delete command deletes or unlists a package from the server. For nuget.org, the action is to unlist the package.

For more details, visit the official website.

Remarks

This is a CLI wrapper with fluent API that allows to modify the following arguments:

  • <packageId> via PackageId
  • <packageVersion> via PackageVersion
  • --api-key via ApiKey
  • --force-english-output via ForceEnglishOutput
  • --interactive via Interactive
  • --no-service-endpoint via NoServiceEndpoint
  • --non-interactive via NonInteractive
  • --source via Source
