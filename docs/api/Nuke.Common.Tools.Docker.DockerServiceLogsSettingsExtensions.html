Remarks

This is a CLI wrapper with fluent API that allows to modify the following arguments:

  • <service> via Service
  • --config via Config
  • --debug via Debug
  • --details via Details
  • --follow via Follow
  • --log-level via LogLevel
  • --no-resolve via NoResolve
  • --no-task-ids via NoTaskIds
  • --no-trunc via NoTrunc
  • --raw via Raw
  • --since via Since
  • --tail via Tail
  • --timestamps via Timestamps
  • --tls via TLS
  • --tlscacert via TLSCaCert
  • --tlscert via TLSCert
  • --tlskey via TLSKey
  • --tlsverify via TLSVerify
