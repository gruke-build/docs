Remarks

This is a CLI wrapper with fluent API that allows to modify the following arguments:

  • <targetPath> via TargetPath
  • -CertificateFingerprint via CertificateFingerprint
  • -CertificatePassword via CertificatePassword
  • -CertificatePath via CertificatePath
  • -CertificateStoreLocation via CertificateStoreLocation
  • -CertificateStoreName via CertificateStoreName
  • -CertificateSubjectName via CertificateSubjectName
  • -ConfigFile via ConfigFile
  • -ForceEnglishOutput via ForceEnglishOutput
  • -HashAlgorithm via HashAlgorithm
  • -NonInteractive via NonInteractive
  • -OutputDirectory via OutputDirectory
  • -Overwrite via Overwrite
  • -Timestamper via Timestamper
  • -TimestampHashAlgorithm via TimestampHashAlgorithm
  • -Verbosity via Verbosity

ResetCertificateSubjectName<T>(T)

Specifies the subject name of the certificate used to search a local certificate store for the certificate. The search is a case-insensitive string comparison using the supplied value, which will find all certificates with the subject name containing that string, regardless of other subject values. The certificate store can be specified by -CertificateStoreName and -CertificateStoreLocation options.
