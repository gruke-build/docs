Class NetlifyDeploySettingsExtensions

Namespace
Nuke.Common.Tools.Netlify
Assembly
Nuke.Common.dll

Netlify’s command line interface (CLI) lets you configure continuous deployment straight from the command line. You can use Netlify CLI to run a local development server that you can share with others, run a local build and plugins, and deploy your site.

For more details, visit the official website.

Remarks

This is a CLI wrapper with fluent API that allows to modify the following arguments:

  • --alias via Alias
  • --auth via Auth
  • --branch via Branch
  • --build via Build
  • --debug via Debug
  • --dir via Directory
  • --functions via Functions
  • --httpProxy via HttpProxy
  • --httpProxyCertificateFilename via HttpProxyCertificateFileName
  • --json via Json
  • --message via Message
  • --open via Open
  • --prod via Production
  • --prodIfUnlocked via ProductionIfUnlocked
  • --site via Site
  • --timeout via Timeout
  • --trigger via Trigger
