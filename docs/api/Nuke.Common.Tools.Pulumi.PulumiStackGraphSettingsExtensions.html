Remarks

This is a CLI wrapper with fluent API that allows to modify the following arguments:

  • <file> via File
  • --color via Color
  • --cwd via Cwd
  • --dependency-edge-color via DependencyEdgeColor
  • --disable-integrity-checking via DisableIntegrityChecking
  • --emoji via Emoji
  • --ignore-dependency-edges via IgnoreDependencyEdges
  • --ignore-parent-edges via IgnoreParentEdges
  • --logflow via LogFlow
  • --logtostderr via LogToStderr
  • --non-interactive via NonInteractive
  • --parent-edge-color via ParentEdgeColor
  • --profiling via Profiling
  • --tracing via Tracing
  • --verbose via Verbose
