Remarks

This is a CLI wrapper with fluent API that allows to modify the following arguments:

  • /AnyType via AnyType
  • /ArrayInstanceType via ArrayInstanceType
  • /ArrayType via ArrayType
  • /DateTimeType via DateTimeType
  • /DictionaryInstanceType via DictionaryInstanceType
  • /DictionaryType via DictionaryType
  • /GenerateOptionalPropertiesAsNullable via GenerateOptionalPropertiesAsNullable
  • /Input via Input
  • /Name via Name
  • /Namespace via Namespace
  • /Output via Output
  • /RequiredPropertiesMustBeDefined via RequiredPropertiesMustBeDefined
  • /ServiceHost via ServiceHost
  • /ServiceSchemes via ServiceSchemes
