Remarks

This is a CLI wrapper with fluent API that allows to modify the following arguments:

  • --config via Config
  • --containerd via Containerd
  • --debug via Debug
  • --engine-image via EngineImage
  • --log-level via LogLevel
  • --registry-prefix via RegistryPrefix
  • --tls via TLS
  • --tlscacert via TLSCaCert
  • --tlscert via TLSCert
  • --tlskey via TLSKey
  • --tlsverify via TLSVerify
  • --version via Version
