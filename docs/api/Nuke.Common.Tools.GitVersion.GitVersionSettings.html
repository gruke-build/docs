Remarks

This is a CLI wrapper with fluent API that allows to modify the following arguments:

  • <targetPath> via TargetPath
  • /b via Branch
  • /c via Commit
  • /diag via Diagnostics
  • /dynamicRepoLocation via DynamicRepositoryLocation
  • /ensureassemblyinfo via EnsureAssemblyInfo
  • /exec via Executable
  • /execargs via ExecutableArguments
  • /l via LogFile
  • /nocache via NoCache
  • /nofetch via NoFetch
  • /output via Output
  • /overrideconfig via ConfigurationOverride
  • /p via Password
  • /proj via MSBuildProject
  • /projargs via MSBuildProjectArguments
  • /showconfig via ShowConfig
  • /showvariable via ShowVariable
  • /u via Username
  • /updateassemblyinfo via UpdateAssemblyInfo
  • /updateassemblyinfofilename via UpdateAssemblyInfoFileNames
  • /url via Url
  • /verbosity via Verbosity
  • /version via Version
