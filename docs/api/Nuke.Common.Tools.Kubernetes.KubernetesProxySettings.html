Class KubernetesProxySettings

Namespace
Nuke.Common.Tools.Kubernetes
Assembly
Nuke.Common.dll

Creates a proxy server or application-level gateway between localhost and the Kubernetes API Server. It also allows serving static content over specified HTTP path. All incoming data enters through one port and gets forwarded to the remote kubernetes API Server port, except for the path matching the static content path.

For more details, visit the official website.

Remarks

This is a CLI wrapper with fluent API that allows to modify the following arguments:

  • --accept-hosts via AcceptHosts
  • --accept-paths via AcceptPaths
  • --address via Address
  • --alsologtostderr via Alsologtostderr
  • --api-prefix via ApiPrefix
  • --as via As
  • --as-group via AsGroup
  • --cache-dir via CacheDir
  • --certificate-authority via CertificateAuthority
  • --client-certificate via ClientCertificate
  • --client-key via ClientKey
  • --cluster via Cluster
  • --context via Context
  • --disable-filter via DisableFilter
  • --insecure-skip-tls-verify via InsecureSkipTlsVerify
  • --kubeconfig via Kubeconfig
  • --log-backtrace-at via LogBacktraceAt
  • --log-dir via LogDir
  • --logtostderr via Logtostderr
  • --match-server-version via MatchServerVersion
  • --namespace via Namespace
  • --port via Port
  • --reject-methods via RejectMethods
  • --reject-paths via RejectPaths
  • --request-timeout via RequestTimeout
  • --server via Server
  • --stderrthreshold via Stderrthreshold
  • --token via Token
  • --unix-socket via UnixSocket
  • --user via User
  • --v via V
  • --vmodule via Vmodule
  • --www via Www
  • --www-prefix via WwwPrefix
