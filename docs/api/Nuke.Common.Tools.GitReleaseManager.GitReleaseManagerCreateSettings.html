Remarks

This is a CLI wrapper with fluent API that allows to modify the following arguments:

  • --assets via AssetPaths
  • --inputFilePath via InputFilePath
  • --logFilePath via LogFilePath
  • --milestone via Milestone
  • --name via Name
  • --owner via RepositoryOwner
  • --password via Password
  • --prerelease via Prerelease
  • --repository via RepositoryName
  • --targetcommitish via TargetCommitish
  • --targetDirectory via TargetDirectory
  • --token via Token
  • --username via UserName
