Class HelmInspectSettings

Namespace
Nuke.Common.Tools.Helm
Assembly
Nuke.Common.dll

This command inspects a chart and displays information. It takes a chart reference ('stable/drupal'), a full path to a directory or packaged chart, or a URL. Inspect prints the contents of the Chart.yaml file and the values.yaml file.

For more details, visit the official website.

Remarks

This is a CLI wrapper with fluent API that allows to modify the following arguments:

  • <chart> via Chart
  • --ca-file via CaFile
  • --cert-file via CertFile
  • --debug via Debug
  • --devel via Devel
  • --home via Home
  • --host via Host
  • --key-file via KeyFile
  • --keyring via Keyring
  • --kube-context via KubeContext
  • --kubeconfig via Kubeconfig
  • --password via Password
  • --repo via Repo
  • --tiller-connection-timeout via TillerConnectionTimeout
  • --tiller-namespace via TillerNamespace
  • --username via Username
  • --verify via Verify
  • --version via Version
