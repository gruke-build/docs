Remarks

This is a CLI wrapper with fluent API that allows to modify the following arguments:

  • /AllowNullableBodyParameters via AllowNullableBodyParameters
  • /AllowReferencesWithProperties via AllowReferencesWithProperties
  • /AspNetCoreEnvironment via AspNetCoreEnvironment
  • /Assembly via Assembly
  • /AssemblyConfig via AssemblyConfig

  • /CreateWebHostBuilderMethod via CreateWebHostBuilderMethod

  • /DocumentProcessors via DocumentProcessors
  • /DocumentTemplate via DocumentTemplate
  • /ExcludedTypeNames via ExcludedTypeNames
  • /FlattenInheritanceHierarchy via FlattenInheritanceHierarchy
  • /GenerateAbstractProperties via GenerateAbstractProperties
  • /GenerateAbstractSchemas via GenerateAbstractSchemas
  • /GenerateEnumMappingDescription via GenerateEnumMappingDescription
  • /GenerateKnownTypes via GenerateKnownTypes
  • /GenerateXmlObjects via GenerateXmlObjects
  • /IgnoreObsoleteProperties via IgnoreObsoleteProperties
  • /InfoDescription via InfoDescription
  • /InfoTitle via InfoTitle
  • /InfoVersion via InfoVersion
  • /OperationProcessors via OperationProcessors
  • /Output via Output
