Remarks

This is a CLI wrapper with fluent API that allows to modify the following arguments:

  • <path> via Path
  • --add-host via AddHost
  • --build-arg via BuildArg
  • --cache-from via CacheFrom
  • --cgroup-parent via CgroupParent
  • --compress via Compress
  • --config via Config
  • --cpu-period via CpuPeriod
  • --cpu-quota via CpuQuota
  • --cpu-shares via CpuShares
  • --cpuset-cpus via CpusetCpus
  • --cpuset-mems via CpusetMems
  • --debug via Debug
  • --disable-content-trust via DisableContentTrust
  • --file via File
  • --force-rm via ForceRm
  • --iidfile via Iidfile
  • --isolation via Isolation
  • --label via Label
  • --log-level via LogLevel
  • --memory via Memory
  • --memory-swap via MemorySwap
  • --network via Network
  • --no-cache via NoCache
  • --output via Output
  • --platform via Platform
  • --progress via Progress
  • --pull via Pull
  • --quiet via Quiet
  • --rm via Rm
  • --secret via Secret
  • --security-opt via SecurityOpt
  • --shm-size via ShmSize
  • --squash via Squash
  • --ssh via Ssh
  • --stream via Stream
  • --tag via Tag
  • --target via Target
  • --tls via TLS
  • --tlscacert via TLSCaCert
  • --tlscert via TLSCert
  • --tlskey via TLSKey
  • --tlsverify via TLSVerify
  • --ulimit via Ulimit
