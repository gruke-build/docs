Remarks

This is a CLI wrapper with fluent API that allows to modify the following arguments:

  • <containers> via Containers
  • --blkio-weight via BlkioWeight
  • --config via Config
  • --cpu-period via CpuPeriod
  • --cpu-quota via CpuQuota
  • --cpu-rt-period via CpuRtPeriod
  • --cpu-rt-runtime via CpuRtRuntime
  • --cpu-shares via CpuShares
  • --cpus via Cpus
  • --cpuset-cpus via CpusetCpus
  • --cpuset-mems via CpusetMems
  • --debug via Debug
  • --kernel-memory via KernelMemory
  • --log-level via LogLevel
  • --memory via Memory
  • --memory-reservation via MemoryReservation
  • --memory-swap via MemorySwap
  • --pids-limit via PidsLimit
  • --restart via Restart
  • --tls via TLS
  • --tlscacert via TLSCaCert
  • --tlscert via TLSCert
  • --tlskey via TLSKey
  • --tlsverify via TLSVerify
