Remarks

This is a CLI wrapper with fluent API that allows to modify the following arguments:

  • <project> via Project
  • --binarylog via BinaryLog
  • --exclude via Exclude
  • --include via Include
  • --include-generated via IncludeGenerated
  • --no-restore via NoRestore
  • --property via Properties
  • --report via Report
  • --severity via Severity
  • --verbosity via Verbosity
  • --verify-no-changes via VerifyNoChanges

Properties

BinaryLog

Logs all project or solution load information to a binary log file.

public string BinaryLog { get; }

Property Value

string

Exclude

A space-separated list of relative file or folder paths to exclude from formatting. The default is none.

public IReadOnlyList<string> Exclude { get; }

Property Value

IReadOnlyList<string>

Include

A space-separated list of relative file or folder paths to include in formatting. The default is all files in the solution or project.

public IReadOnlyList<string> Include { get; }

Property Value

IReadOnlyList<string>

IncludeGenerated

Formats files generated by the SDK.

public bool? IncludeGenerated { get; }

Property Value

bool?

NoRestore

Doesn't execute an implicit restore before formatting. Default is to do implicit restore.

public bool? NoRestore { get; }

Property Value

bool?

Project

The MSBuild project or solution to run code formatting on. If a project or solution file is not specified, MSBuild searches the current working directory for a file that has a file extension that ends in proj or sln, and uses that file.

public string Project { get; }

Property Value

string

Properties

Set or override the specified project-level properties, where name is the property name and value is the property value. Specify each property separately, or use a semicolon or comma to separate multiple properties, as the following example shows:

--property:WarningLevel=2;OutDir=bin\Debug

public IReadOnlyDictionary<string, object> Properties { get; }

Property Value

IReadOnlyDictionary<string, object>

Report

Produces a JSON report in the directory specified by REPORT_PATH.

public string Report { get; }

Property Value

string

Severity

The minimum severity of diagnostics to fix. Allowed values are info, warn, and error. The default value is warn.

public DotNetFormatSeverity Severity { get; }

Property Value

DotNetFormatSeverity

Verbosity

Sets the verbosity level. Allowed levels are q[uiet], m[inimal], n[ormal], d[etailed], and diag[nostic].

public DotNetVerbosity Verbosity { get; }

Property Value

DotNetVerbosity

VerifyNoChanges

Verifies that no formatting changes would be performed. Terminates with a non zero exit code if any files would have been formatted.

public bool? VerifyNoChanges { get; }

bool?
