Remarks

This is a CLI wrapper with fluent API that allows to modify the following arguments:

  • <service> via Service
  • --args via Args
  • --config via Config
  • --config-add via ConfigAdd
  • --config-rm via ConfigRm
  • --constraint-add via ConstraintAdd
  • --constraint-rm via ConstraintRm
  • --container-label-add via ContainerLabelAdd
  • --container-label-rm via ContainerLabelRm
  • --credential-spec via CredentialSpec
  • --debug via Debug
  • --detach via Detach
  • --dns-add via DnsAdd
  • --dns-option-add via DnsOptionAdd
  • --dns-option-rm via DnsOptionRm
  • --dns-rm via DnsRm
  • --dns-search-add via DnsSearchAdd
