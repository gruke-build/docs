ParameterAttribute

Injected parameters are resolved case-insensitively in the following order:

  • From command-line arguments (e.g., -arg value)
  • From environment variables (e.g., Arg=value)

For value-types, there is a distinction between pure value-types, and their nullable counterparts. For instance, int will have its default value 0 even when it's not supplied via command-line or environment variable.
