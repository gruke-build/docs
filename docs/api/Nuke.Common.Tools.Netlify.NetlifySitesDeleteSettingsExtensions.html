Remarks

This is a CLI wrapper with fluent API that allows to modify the following arguments:

  • <siteId> via SiteId
  • --debug via Debug
  • --force via Force
  • --httpProxy via HttpProxy
  • --httpProxyCertificateFilename via HttpProxyCertificateFileName
