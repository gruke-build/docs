Remarks

This is a CLI wrapper with fluent API that allows to modify the following arguments:

  • --color via Color
  • --config via Config
  • --config-file via ConfigFile
  • --config-path via ConfigPath
  • --cwd via Cwd
  • --debug via Debug
  • --diff via Diff
  • --disable-integrity-checking via DisableIntegrityChecking
  • --emoji via Emoji
  • --expect-no-changes via ExpectNoChanges
  • --logflow via LogFlow
  • --logtostderr via LogToStderr
  • --message via Message
  • --non-interactive via NonInteractive
  • --parallel via Parallel
  • --policy-pack via PolicyPack
  • --policy-pack-config via PolicyPackConfig
  • --profiling via Profiling
  • --refresh via Refresh
  • --replace via Replace
  • --secrets-provider via SecretsProvider
  • --show-config via ShowConfig
  • --show-reads via ShowReads
  • --show-replacement-steps via ShowReplacementSteps
  • --show-sames via ShowSames
  • --skip-preview via SkipPreview
  • --stack via Stack
  • --suppress-outputs via SuppressOutputs
  • --suppress-permalink via SuppressPermalink
  • --target via Target
  • --target-dependents via TargetDependents
  • --target-replace via TargetReplace
  • --tracing via Tracing
  • --verbose via Verbose
  • --yes via Yes
