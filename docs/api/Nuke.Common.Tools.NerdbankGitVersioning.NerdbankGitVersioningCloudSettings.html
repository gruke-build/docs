Remarks

This is a CLI wrapper with fluent API that allows to modify the following arguments:

  • --all-vars via AllVars
  • --ci-system via CISystem
  • --common-vars via CommonVars
  • --define via Variables
  • --metadata via Metadata
  • --project via Project
  • --version via Version
