Remarks

This is a CLI wrapper with fluent API that allows to modify the following arguments:

  • <assemblyFile> via AssemblyFile
  • --affinity via Affinity
  • --allCategories via AllCategories
  • --allStats via DisplayAllStatistics
  • --anyCategories via AnyCategories
  • --artifacts via ArtifactsDirecory
  • --attribute via AttributeNames
  • --buildTimeout via BuildTimeout
  • --cli via CliPath
  • --clrVersion via ClrVersion
  • --coreRtVersion via CoreRtVersion
  • --coreRun via CoreRunPaths
  • --counters via HardwareCounters
  • --disableLogFile via DisableLogFile
  • --disasm via Disassembly
  • --disasmDepth via DisassemblyRecursiveDepth
  • --disasmDiff via DisassemblyDiff
  • --exporters via Exporters
  • --filter via Filter
  • --ilcPath via CoreRtPath
  • --info via PrintInformation
  • --inProcess via RunInProcess
  • --invocationCount via InvocationCount
  • --iterationCount via IterationCount
  • --iterationTime via IterationTime
  • --job via Job
  • --join via Join
  • --keepFiles via KeepBenchmarkFiles
  • --launchCount via LaunchCount
  • --list via ListBenchmarkCaseMode
  • --maxIterationCount via MaxIterationCount
  • --maxWarmupCount via MaxWarmupCount
  • --maxWidth via MaxParameterColumnWidth
  • --memory via MemoryStats
  • --minIterationCount via MinIterationCount
  • --minWarmupCount via MinWarmupCount
  • --monoPath via MonoPath
  • --noOverwrite via DontOverwriteResults
  • --outliers via OutlierMode
  • --packages via RestorePath
  • --profiler via Profiler
  • --runOncePerIteration via RunOncePerIteration
  • --runtimes via Runtimes
  • --statisticalTest via StatisticalTestThreshold
  • --stopOnFirstError via StopOnFirstError
  • --strategy via RunStrategy
  • --threading via ThreadingStats
  • --unrollFactor via UnrollFactor
  • --warmupCount via WarmupCount
