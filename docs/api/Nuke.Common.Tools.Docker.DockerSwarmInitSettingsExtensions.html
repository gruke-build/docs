Remarks

This is a CLI wrapper with fluent API that allows to modify the following arguments:

  • --advertise-addr via AdvertiseAddr
  • --autolock via Autolock
  • --availability via Availability
  • --cert-expiry via CertExpiry
  • --config via Config
  • --data-path-addr via DataPathAddr
  • --data-path-port via DataPathPort
  • --debug via Debug
  • --default-addr-pool via DefaultAddrPool
  • --default-addr-pool-mask-length via DefaultAddrPoolMaskLength
  • --dispatcher-heartbeat via DispatcherHeartbeat
  • --external-ca via ExternalCa
  • --force-new-cluster via ForceNewCluster
  • --listen-addr via ListenAddr
  • --log-level via LogLevel
  • --max-snapshots via MaxSnapshots
  • --snapshot-interval via SnapshotInterval
  • --task-history-limit via TaskHistoryLimit
  • --tls via TLS
  • --tlscacert via TLSCaCert
  • --tlscert via TLSCert
  • --tlskey via TLSKey
  • --tlsverify via TLSVerify
