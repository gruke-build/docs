Remarks

This is a CLI wrapper with fluent API that allows to modify the following arguments:

  • <volume> via Volume
  • --config via Config
  • --debug via Debug
  • --driver via Driver
  • --label via Label
  • --log-level via LogLevel
  • --name via Name
  • --opt via Opt
  • --tls via TLS
  • --tlscacert via TLSCaCert
  • --tlscert via TLSCert
  • --tlskey via TLSKey
  • --tlsverify via TLSVerify
