Class EntityFrameworkDbContextScaffoldSettingsExtensions

Namespace
Nuke.Common.Tools.EntityFramework
Assembly
Nuke.Common.dll

The dotnet-ef dbcontext scaffold command is used to generate code for a DbContext and entity types for a database. In order for this command to generate an entity type, the database table must have a primary key.

For more details, visit the official website.

[PublicAPI]

Remarks

This is a CLI wrapper with fluent API that allows to modify the following arguments:

  • <connection> via Connection
  • <provider> via Provider
  • --configuration via Configuration
  • --context via Context
  • --context-dir via ContextDirectory
  • --context-namespace via ContextNamespace
  • --data-annotations via DataAnnotations
  • --force via Force
  • --framework via Framework
  • --json via Json
  • --namespace via Namespace
  • --no-build via NoBuild
  • --no-color via NoColor
  • --no-onconfiguring via NoOnConfiguring
  • --no-pluralize via NoPluralize
  • --output-dir via OutputDirectory
  • --prefix-output via PrefixOutput
  • --project via Project
  • --runtime via Runtime
  • --schema via Schemas
  • --startup-project via StartupProject
  • --table via Tables
  • --use-database-names via UseDatabaseNames
  • --verbose via Verbose
