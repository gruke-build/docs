Remarks

This is a CLI wrapper with fluent API that allows to modify the following arguments:

  • --baseUrl via BaseUrl
  • --bootstrapperExe via BootstrapperExecutable
  • --checkForUpdate via CheckForUpdate
  • --createShortcut via CreateShortcut
  • --download via Download
  • --framework-version via FrameworkVersion
  • --icon via Icon
  • --install via Install
  • --loadingGif via LoadingGif
  • --no-delta via GenerateNoDelta
  • --no-msi via GenerateNoMsi
  • --packagesDir via PackagesDirectory
  • --process-start-args via ProcessStartArguments
  • --processStart via ProcessStart
  • --processStartAndWait via ProcessStartAndWait
  • --releaseDir via ReleaseDirectory
  • --releasify via Releasify
  • --removeShortcut via RemoveShortcut
  • --setupIcon via SetupIcon
  • --shortcut-locations via ShortcutLocations
  • --signWithParams via SignWithParameters
  • --uninstall via Uninstall
  • --update via Update
  • --updateSelf via UpdateSelf
