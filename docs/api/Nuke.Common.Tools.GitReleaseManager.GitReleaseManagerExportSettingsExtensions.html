Remarks

This is a CLI wrapper with fluent API that allows to modify the following arguments:

  • --fileOutputPath via FileOutputPath
  • --logFilePath via LogFilePath
  • --owner via RepositoryOwner
  • --password via Password
  • --repository via RepositoryName
  • --tagName via TagName
  • --targetDirectory via TargetDirectory
  • --token via Token
  • --username via UserName
