Remarks

This is a CLI wrapper with fluent API that allows to modify the following arguments:

  • <nodes> via Nodes
  • --config via Config
  • --debug via Debug
  • --force via Force
  • --log-level via LogLevel
  • --tls via TLS
  • --tlscacert via TLSCaCert
  • --tlscert via TLSCert
  • --tlskey via TLSKey
  • --tlsverify via TLSVerify
