Class HelmLintSettingsExtensions

Namespace
Nuke.Common.Tools.Helm
Assembly
Nuke.Common.dll

This command takes a path to a chart and runs a series of tests to verify that the chart is well-formed. If the linter encounters things that will cause the chart to fail installation, it will emit [ERROR] messages. If it encounters issues that break with convention or recommendation, it will emit [WARNING] messages.

For more details, visit the official website.

Remarks

This is a CLI wrapper with fluent API that allows to modify the following arguments:

  • <path> via Path
  • --debug via Debug
  • --home via Home
  • --host via Host
  • --kube-context via KubeContext
  • --kubeconfig via Kubeconfig
  • --namespace via Namespace
  • --set via Set
  • --set-file via SetFile
  • --set-string via SetString
  • --strict via Strict
  • --tiller-connection-timeout via TillerConnectionTimeout
  • --tiller-namespace via TillerNamespace
  • --values via Values
