Class HelmResetSettings

Namespace
Nuke.Common.Tools.Helm
Assembly
Nuke.Common.dll

This command uninstalls Tiller (the Helm server-side component) from your Kubernetes Cluster and optionally deletes local configuration in $HELM_HOME (default ~/.helm/).

For more details, visit the official website.

Remarks

This is a CLI wrapper with fluent API that allows to modify the following arguments:

  • --debug via Debug
  • --force via Force
  • --home via Home
  • --host via Host
  • --kube-context via KubeContext
  • --kubeconfig via Kubeconfig
  • --remove-helm-home via RemoveHelmHome
  • --tiller-connection-timeout via TillerConnectionTimeout
  • --tiller-namespace via TillerNamespace
  • --tls via Tls
  • --tls-ca-cert via TlsCaCert
  • --tls-cert via TlsCert
  • --tls-hostname via TlsHostname
  • --tls-key via TlsKey
  • --tls-verify via TlsVerify
