ILRepack is meant at replacing ILMerge / Mono.Merge.

The former being closed-source (now open-sourced), impossible to customize, slow, resource consuming and many more. The later being deprecated, unsupported, and based on an old version of Mono.Cecil.

Here we're using latest (slightly modified) Cecil sources (0.9), you can find the fork here. Mono.Posix is also required (build only, it gets merged afterwards) for executable bit set on target file.

For more details, visit the official website.

Remarks

This is a CLI wrapper with fluent API that allows to modify the following arguments:

  • <assemblies> via Assemblies
  • --allowdup via AllowDuplicate
  • --allowduplicateresources via AllowDuplicateResources
  • --allowMultiple via AllowMultiple
  • --attr via Attr
  • --copyattrs via CopyAttributes
  • --delaysign via DelaySign
  • --internalize via Internalize
  • --internalize via InternalizeExcludeFile
  • --keycontainer via KeyContainer
  • --keyfile via KeyFile
  • --lib via Lib
  • --log via LogFile
  • --ndebug via NoDebug
  • --out via Output
  • --parallel via Parallel
  • --pause via Pause
  • --renameInternalized via RenameInternalized
  • --repackdrop:AttributeClass via RepackDrop
  • --target via Target
  • --targetplatform via TargetPlatform
  • --union via Union
  • --ver via Version
  • --verbose via Verbose
  • --wildcards via Wildcards
  • --xmldocs via Xmldocs
  • --zeropekind via ZeroPekind
