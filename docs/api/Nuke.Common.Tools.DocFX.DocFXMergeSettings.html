Remarks

This is a CLI wrapper with fluent API that allows to modify the following arguments:

  • <configFile> via ConfigFile
  • --content via Content
  • --correlationId via CorrelationId
  • --fileMetadataFile via FileMetadataFilePath
  • --globalMetadata via GlobalMetadata
  • --globalMetadataFile via GlobalMetadataFilePath
  • --help via PrintHelpMessage
  • --log via LogFilePath
  • --logLevel via LogLevel
  • --repositoryRoot via RepoRoot
  • --tocMetadata via TocMetadata
  • --warningsAsErrors via WarningsAsErrors
