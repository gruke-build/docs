Remarks

This is a CLI wrapper with fluent API that allows to modify the following arguments:

  • -- via Properties
  • --accept-license via AcceptLicense
  • --allow-unofficial-build via AllowUnofficialBuild
  • --automaticpackage via AutomaticPackage
  • --build-package via BuildPackage
  • --cache-location via CacheLocation
  • --checksum via Checksum
  • --checksum-type via ChecksumType
  • --checksum-x64 via Checksum64
  • --confirm via Confirm
  • --debug via Debug
  • --execution-timeout via CommandExecutionTimeout
  • --fail-on-standard-error via FailOnStandardError
  • --file via LocationOfBinary
  • --file64 via LocationOfBinary64
  • --force via Force
  • --from-programs-and-features via GenerateFromInstalledSoftware
  • --generate-for-community via GenerateForCommunity
  • --include-architecture-in-name via IncludeArchitectureInName
  • --limit-output via LimitOutput
  • --log-file via LogFile
  • --maintainer via Maintainer
  • --name via Name
  • --no-color via NoColor
  • --no-progress via DoNotShowProgress
  • --output-directory via OutputDirectory
  • --pause-on-error via PauseOnError
  • --proxy via Proxy
  • --proxy-bypass-list via ProxyBypassList
  • --proxy-bypass-on-local via ProxyBypassOnLocal
  • --proxy-password via ProxyPassword
  • --proxy-user via ProxyUserName
  • --remove-architecture-from-name via RemoveArchitectureFromName
  • --template-name via TemplateName
  • --trace via Trace
  • --use-built-in-template via BuildInTemplate
  • --use-original-files-location via UseOriginalFilesLocation
  • --use-system-powershell via UseSystemPowerShell
  • --verbose via Verbose
  • --version via Version
