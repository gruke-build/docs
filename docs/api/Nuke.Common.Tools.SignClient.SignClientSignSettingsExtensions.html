Remarks

This is a CLI wrapper with fluent API that allows to modify the following arguments:

  • --baseDirectory via BaseDirectory
  • --config via Config
  • --description via Description
  • --descriptionUrl via DescriptionUrl
  • --fileList via FileList
  • --input via Input
  • --maxConcurrency via MaxConcurrency
  • --name via Name
  • --output via Output
  • --secret via Secret
  • --user via Username
