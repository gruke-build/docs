  • <assemblies> via Assemblies
  • --appveyor via AppVeyor
  • --html via HtmlOutput
  • --no-appveyor-autodetect via NoAppVeyor
  • --no-color via NoColor
  • --no-teamcity-autodetect via NoTeamCity
