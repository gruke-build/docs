Remarks

This is a CLI wrapper with fluent API that allows to modify the following arguments:

  • --configuration via Configuration
  • --context via Context
  • --framework via Framework
  • --json via Json
  • --no-build via NoBuild
  • --no-color via NoColor
  • --output via Output
  • --prefix-output via PrefixOutput
  • --project via Project
  • --runtime via Runtime
  • --startup-project via StartupProject
  • --verbose via Verbose
