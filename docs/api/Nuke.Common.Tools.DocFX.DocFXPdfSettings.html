Remarks

This is a CLI wrapper with fluent API that allows to modify the following arguments:

  • <configFile> via ConfigFile
  • --basePath via BasePath
  • --changesFile via ChangesFile
  • --cleanupCacheHistory via CleanupCacheHistory
  • --content via Content
  • --correlationId via CorrelationId
  • --css via CssFilePath
  • --debug via EnableDebugMode
  • --debugOutput via OutputFolderForDebugFiles
  • --disableGitFeatures via DisableGitFeatures
  • --dryRun via DryRun
  • --errorHandling via LoadErrorHandling
  • --excludedTocs via ExcludedTocs
  • --exportRawModel via ExportRawModel
  • --exportViewModel via ExportViewModel
  • --falName via FALName
  • --fileMetadataFile via FileMetadataFilePath
  • --fileMetadataFiles via FileMetadataFilePaths
  • --force via ForceRebuild
  • --forcePostProcess via ForcePostProcess
  • --generatesAppendices via GeneratesAppendices
  • --generatesExternalLink via GeneratesExternalLink
  • --globalMetadata via GlobalMetadata
  • --globalMetadataFile via GlobalMetadataFilePath
  • --globalMetadataFiles via GlobalMetadataFilePaths
  • --help via PrintHelpMessage
  • --host via PdfHost
  • --hostname via Host
  • --intermediateFolder via IntermediateFolder
  • --keepFileLink via KeepFileLink
  • --keepRawFiles via KeepRawFiles
  • --locale via Locale
  • --log via LogFilePath
  • --logLevel via LogLevel
  • --lruSize via LruSize
  • --markdownEngineName via MarkdownEngineName
  • --markdownEngineProperties via MarkdownEngineProperties
  • --maxParallelism via MaxParallelism
  • --name via Name
  • --noLangKeyword via NoLangKeyword
  • --noStdin via NoInputStreamArgs
  • --output via OutputFolder
  • --overwrite via Overwrite
  • --port via Port
  • --postProcessors via PostProcessors
  • --rawModelOutputFolder via RawModelOutputFolder
  • --rawOutputFolder via RawOutputFolder
  • --repositoryRoot via RepoRoot
  • --resource via Resource
  • --schemaLicense via SchemaLicense
  • --serve via Serve
  • --template via Templates
  • --theme via Themes
  • --viewModelOutputFolder via ViewModelOutputFolder
  • --warningsAsErrors via WarningsAsErrors
  • --xref via XRefMaps
