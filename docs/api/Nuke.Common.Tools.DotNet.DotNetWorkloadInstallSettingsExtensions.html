Class DotNetWorkloadInstallSettingsExtensions

Namespace
Nuke.Common.Tools.DotNet
Assembly
Nuke.Common.dll

The dotnet workload install command installs one or more optional workloads. Optional workloads can be installed on top of the .NET SDK to provide support for various application types, such as .NET MAUI and Blazor WebAssembly AOT.

For more details, visit the official website.

Remarks

This is a CLI wrapper with fluent API that allows to modify the following arguments:

  • <workloadId> via WorkloadId
  • --configFile via ConfigFile
  • --disable-parallel via DisableParallel
  • --ignore-failed-sources via IgnoreFailedSources
  • --include-previews via IncludePreviews
  • --interactive via Interactive
  • --no-cache via NoCache
  • --skip-manifest-update via SkipManifestUpdate
  • --skip-sign-check via SkipSignCheck
  • --source via Source
  • --temp-dir via TempDir
  • --verbosity via Verbosity
