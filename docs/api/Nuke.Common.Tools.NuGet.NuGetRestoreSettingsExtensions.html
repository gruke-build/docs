Remarks

This is a CLI wrapper with fluent API that allows to modify the following arguments:

  • <targetPath> via TargetPath
  • -ConfigFile via ConfigFile
  • -DirectDownload via DirectDownload
  • -DisableParallelProcessing via DisableParallelProcessing
  • -FallbackSource via FallbackSource
  • -ForceEnglishOutput via ForceEnglishOutput
  • -MSBuildPath via MSBuildPath
  • -MSBuildVersion via MSBuildVersion
  • -NoCache via NoCache
  • -NonInteractive via NonInteractive
  • -OutputDirectory via OutputDirectory
  • -PackageSaveMode via PackageSaveMode
  • -PackagesDirectory via PackagesDirectory
  • -Project2ProjectTimeOut via Project2ProjectTimeOut
  • -Recursive via Recursive
  • -RequireConsent via RequireConsent
  • -SolutionDirectory via SolutionDirectory
  • -Source via Source
  • -Verbosity via Verbosity
