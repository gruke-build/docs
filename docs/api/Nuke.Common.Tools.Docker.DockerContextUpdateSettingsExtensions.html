Remarks

This is a CLI wrapper with fluent API that allows to modify the following arguments:

  • <context> via Context
  • --config via Config
  • --debug via Debug
  • --default-stack-orchestrator via DefaultStackOrchestrator
  • --description via Description
  • --docker via Docker
  • --kubernetes via Kubernetes
  • --log-level via LogLevel
  • --tls via TLS
  • --tlscacert via TLSCaCert
  • --tlscert via TLSCert
  • --tlskey via TLSKey
  • --tlsverify via TLSVerify
