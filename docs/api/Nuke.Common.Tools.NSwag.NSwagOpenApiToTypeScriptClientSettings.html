Remarks

This is a CLI wrapper with fluent API that allows to modify the following arguments:

  • /EnumNameGeneratorType via EnumNameGeneratorType
  • /Input via Input
  • /Output via Output
  • /PropertyNameGeneratorType via PropertyNameGeneratorType
  • /ServiceHost via ServiceHost
  • /ServiceSchemes via ServiceSchemes
  • /TemplateDirectory via TemplateDirectory
  • /TypeNameGenerator via TypeNameGenerator
