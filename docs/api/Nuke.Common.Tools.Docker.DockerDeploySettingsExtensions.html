Remarks

This is a CLI wrapper with fluent API that allows to modify the following arguments:

  • <stack> via Stack
  • --bundle-file via BundleFile
  • --compose-file via ComposeFile
  • --config via Config
  • --debug via Debug
  • --log-level via LogLevel
  • --namespace via Namespace
  • --prune via Prune
  • --resolve-image via ResolveImage
  • --tls via TLS
  • --tlscacert via TLSCaCert
  • --tlscert via TLSCert
  • --tlskey via TLSKey
  • --tlsverify via TLSVerify
  • --with-registry-auth via WithRegistryAuth
