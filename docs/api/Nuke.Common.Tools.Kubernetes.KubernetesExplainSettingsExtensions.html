Namespace
Nuke.Common.Tools.Kubernetes
Assembly
Nuke.Common.dll

List the fields for supported resources This command describes the fields associated with each supported API resource. Fields are identified via a simple JSONPath identifier: <type>.<fieldName>[.<fieldName>] Add the --recursive flag to display all of the fields at once without descriptions. Information about each field is retrieved from the server in OpenAPI format.Use "kubectl api-resources" for a complete list of supported resources.

For more details, visit the official website.

Remarks

This is a CLI wrapper with fluent API that allows to modify the following arguments:

  • <resource> via Resource
  • --alsologtostderr via Alsologtostderr
  • --api-version via ApiVersion
  • --as via As
  • --as-group via AsGroup
  • --cache-dir via CacheDir
  • --certificate-authority via CertificateAuthority
  • --client-certificate via ClientCertificate
  • --client-key via ClientKey
  • --cluster via Cluster
  • --context via Context
  • --insecure-skip-tls-verify via InsecureSkipTlsVerify
  • --kubeconfig via Kubeconfig
  • --log-backtrace-at via LogBacktraceAt
  • --log-dir via LogDir
  • --logtostderr via Logtostderr
  • --match-server-version via MatchServerVersion
  • --namespace via Namespace
  • --recursive via Recursive
  • --request-timeout via RequestTimeout
  • --server via Server
  • --stderrthreshold via Stderrthreshold
  • --token via Token
  • --user via User
  • --v via V
  • --vmodule via Vmodule
