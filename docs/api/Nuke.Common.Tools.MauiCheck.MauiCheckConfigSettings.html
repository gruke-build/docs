Remarks

This is a CLI wrapper with fluent API that allows to modify the following arguments:

  • --dotnet-pre via DotNetPrerelease
  • --dotnet-rollForward via DotNetRollForward
  • --dotnet-version via DotNetVersion
  • --nuget-sources via NuGetSources
