Remarks

This is a CLI wrapper with fluent API that allows to modify the following arguments:

  • <scriptFile> via ScriptFile
  • /D via DefinedSymbols
  • /INPUTCHARSET via InputCharset
  • /NOCD via NoCurrentDirectoryChange
  • /NOCONFIG via NoConfig
  • /O via LogOutputFile
  • /OUTPUTCHARSET via OutputCharset
  • /P via CompilerPriority
  • /PPO via OnlyRunPreprocessor
  • /SAFEPPO via OnlyRunSaveVersionPreprocessor
  • /V via Verbosity
  • /WX via TreatWarningsAsErrors
  • /X via PreExecutionCodes
  • /X via PostExecutionCodes
