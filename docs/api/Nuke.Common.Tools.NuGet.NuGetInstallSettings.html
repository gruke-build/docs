Remarks

This is a CLI wrapper with fluent API that allows to modify the following arguments:

  • <packageID> via PackageID
  • -ConfigFile via ConfigFile
  • -DependencyVersion via DependencyVersion
  • -DisableParallelProcessing via DisableParallelProcessing
  • -ExcludeVersion via ExcludeVersion
  • -FallbackSource via FallbackSource
  • -ForceEnglishOutput via ForceEnglishOutput
  • -Framework via Framework
  • -NoCache via NoCache
  • -NonInteractive via NonInteractive
  • -OutputDirectory via OutputDirectory
  • -PackageSaveMode via PackageSaveMode
  • -PreRelease via PreRelease
  • -RequireConsent via RequireConsent
  • -SolutionDirectory via SolutionDirectory
  • -Source via Source
  • -Verbosity via Verbosity
  • -Version via Version

ConfigFile

The NuGet configuration file to apply. If not specified, %AppData%\NuGet\NuGet.Config (Windows) or ~/.nuget/NuGet/NuGet.Config (Mac/Linux) is used.

public string ConfigFile { get; }

Property Value

string

DependencyVersion

(4.4+) The version of the dependency packages to use, which can be one of the following:

  • Lowest: (default) the lowest version.
  • HighestPatch: the version with the lowest major, lowest minor, highest patch.
  • HighestMinor: the version with the lowest major, highest minor, highest patch.
  • Highest: the highest version.
  • Ignore: No dependency packages will be used.
public DependencyVersion DependencyVersion { get; }
