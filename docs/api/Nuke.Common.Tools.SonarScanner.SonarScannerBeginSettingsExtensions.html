Class SonarScannerBeginSettingsExtensions

Namespace
Nuke.Common.Tools.SonarScanner
Assembly
Nuke.Common.dll

The SonarScanner for MSBuild is the recommended way to launch a SonarQube or SonarCloud analysis for projects/solutions using MSBuild or dotnet command as build tool.

For more details, visit the official website.

Remarks

This is a CLI wrapper with fluent API that allows to modify the following arguments:

  • /d via AdditionalParameters
  • /d:sonar.analysis. via AdditionalAnalysisParameters
  • /d:sonar.branch.name via BranchName
  • /d:sonar.clientcert.password via ClientCertificatePassword
  • /d:sonar.clientcert.path via ClientCertificatePath
  • /d:sonar.coverage.exclusions via CoverageExclusions
  • /d:sonar.coverageReportPaths via GenericCoveragePaths
  • /d:sonar.cpd.exclusions via DuplicationExclusions
  • /d:sonar.cs.dotcover.reportsPaths via DotCoverPaths
  • /d:sonar.cs.nunit.reportsPaths via NUnitTestReports
  • /d:sonar.cs.opencover.reportsPaths via OpenCoverPaths
  • /d:sonar.cs.vscoveragexml.reportsPaths via VisualStudioCoveragePaths
  • /d:sonar.cs.vstest.reportsPaths via VSTestReports
  • /d:sonar.cs.xunit.reportsPaths via XUnitTestReports
  • /d:sonar.dotnet.excludeTestProjects via ExcludeTestProjects
  • /d:sonar.exclusions via SourceExclusions
  • /d:sonar.host.url via Server
  • /d:sonar.inclusions via SourceInclusions
  • /d:sonar.links.ci via ContinuousIntegrationUrl
  • /d:sonar.links.homepage via Homepage
  • /d:sonar.links.issue via IssueTrackerUrl
  • /d:sonar.links.scm via SCMUrl
  • /d:sonar.login via Login
  • /d:sonar.password via Password
  • /d:sonar.projectDescription via Description
  • /d:sonar.pullrequest.base via PullRequestBase
  • /d:sonar.pullrequest.branch via PullRequestBranch
  • /d:sonar.pullrequest.key via PullRequestKey
  • /d:sonar.qualitygate.timeout via QualityGateTimeout
  • /d:sonar.qualitygate.wait via QualityGateWait
  • /d:sonar.scm.exclusions.disabled via ScmExclusionsDisabled
  • /d:sonar.scm.forceReloadAll via ScmForceReloadAll
  • /d:sonar.scm.provider via ScmProvider
  • /d:sonar.scm.revision via ScmRevision
  • /d:sonar.sourceEncoding via SourceEncoding
  • /d:sonar.test.exclusions via TestFileExclusions
  • /d:sonar.test.inclusions via TestFileInclusions
  • /d:sonar.token via Token
  • /d:sonar.verbose via Verbose
  • /d:sonar.ws.timeout via WebServiceTimeout
  • /k via ProjectKey
  • /n via Name
  • /o via Organization
  • /v via Version
