Remarks

This is a CLI wrapper with fluent API that allows to modify the following arguments:

  • <configuration> via Configuration
  • --AllowSymbolServerAccess via AllowSymbolServerAccess
  • --AnalyseTargetArguments via AnalyseTargetArguments
  • --AttributeFilters via AttributeFilters
  • --DisableDefaultFilters via DisableDefaultFilters
  • --Filters via Filters
  • --HideAutoProperties via HideAutoProperties
  • --InheritConsole via InheritConsole
  • --LogFile via LogFile
  • --Output via OutputFile
  • --ProcessFilters via ProcessFilters
  • --ReportType via ReportType
  • --ReturnTargetExitCode via ReturnTargetExitCode
  • --Scope via Scope
  • --SymbolSearchPaths via SymbolSearchPaths
  • --TargetArguments via TargetArguments
  • --TargetExecutable via TargetExecutable
  • --TargetWorkingDir via TargetWorkingDirectory
  • --TempDir via TempDirectory
